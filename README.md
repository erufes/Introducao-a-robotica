# Introducao-a-robotica
Este repositório é de domínio público para divulgação, a fim de fornecer um material iniciante e prático para que pessoas interessadas em robótica iniciem seus próprios trabalhos.


## 🛠️ Materiais e Custos Estimados (Valores para Abril de 2026)

O custo total estimado para o kit geral básico é de **por volta de R$ 200**, incluindo frete e impostos (via AliExpress).

* **Kit inicial ESP32:** Por volta de R$ 80,00 no AliExpress.
* **Servo Motor:** Por volta de R$ 9,00 cada no AliExpress.
* **Baterias e Suporte:** Por volta de R$ 20,00 no AliExpress.
  * *Nota:* As mais usadas são as baterias de lítio 18650 ("baterias de lanterna tática"). Duas unidades são suficientes para iniciantes. É necessário comprar também uma case (suporte) para 2 baterias de lítio.
* **Kit de Solda:** Por volta de R$ 30,00 no AliExpress.

> 💡 **Dica de Sustentabilidade e Economia (Sucateamento):** Baterias, estruturas e outros componentes para os projetos podem ser reaproveitados de eletrônicos velhos que não são mais usados. Exemplos práticos explorados mais à frente no guia incluem: retirar baterias de fones Bluetooth, extrair baterias de lítio de laptops antigos e transformar uma fonte de desktop velha em uma fonte de corrente e tensão contínua de bancada para testes.
