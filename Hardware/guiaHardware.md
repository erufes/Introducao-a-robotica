Os componentes base de robótica se encontram dentro das áreas de microcontroladores, carga (motores) e sensores.
 Para iniciar ou dar continuação em algum projeto, é importante sempre ter os datasheets dos componentes e dos
 modelos destes que serão usados no seu projeto. Nos datasheets, os fabricantes inserem todas as informações dos 
componentes: tensão máxima, corrente máxima, qual sinal é esperado dos sensores (sinais são dados digitais ou 
analógicos em forma de onda contendo as informações que os sensores tiram do ambiente externo) e, às vezes, até 
diagramas de circuitos recomendados para o melhor funcionamento do componente.


Sucateamento: Geralmente a maior parte dos dispositivos e possivel reutilizar as baterias para outros projetos 
e/ou o circuito de carregamento. Outros componentes são mais dificeis de serem readaptados para ultilização, mas
não impossiveis.
